## Tech Stack

**Frontend:**
- Next.js 14 (App Router)
- shadcn/ui components (Card, Button, Badge, Checkbox)
- Tailwind CSS for styling
- TypeScript

**Backend:**
- Next.js API routes (/api/todos)
- In-memory database (todos array for simplicity - can upgrade to PostgreSQL later)

**Features:**
- Add todo (POST /api/todos)
- List todos (GET /api/todos)
- Update todo (PUT /api/todos/:id)
- Delete todo (DELETE /api/todos/:id)
- Toggle status (PATCH /api/todos/:id/toggle)
- Filter by status (all/active/completed)
- Filter by priority (high/medium/low)
- Sort by newest/oldest
